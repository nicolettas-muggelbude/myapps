# Screenshots für MyApps

- GitHub README
- Dokumentation

### 1. `main-window.png` (Pflicht für Flathub)
- Hauptfenster mit Listenansicht
- Zeigt Icons, deutsche Beschreibungen, Suchleiste
- ~10-20 Apps sichtbar
- Empfohlene Größe: 1200x850px

### 2. `table-view.png` (Optional)
- Tabellenansicht
- Alle Spalten sichtbar
- Zeigt Performance mit vielen Apps

### 3. `search-demo.png` (Optional)
- Suchfunktion in Aktion
- Suchfeld mit Eingabe
- Gefilterte Ergebnisse

### 4. `dark-mode.png` (Optional)
- Dark Mode aktiv
- Zeigt Libadwaita-Integration

## Format:
- PNG (bevorzugt)
- Komprimiert mit pngquant/optipng
- Max 1 MB pro Screenshot
